Треним комит и пуш

Git Commit — это команда в системе контроля версий Git, которая фиксирует изменения в репозитории. Когда пользователь использует её, создаётся «снимок» текущего состояния проекта, включая все внесённые изменения.

Git push — команда для отправки изменений из локального репозитория в удалённый. Она облегчает процесс синхронизации в команде разработчиков, пересылая совершённые апдейты.
